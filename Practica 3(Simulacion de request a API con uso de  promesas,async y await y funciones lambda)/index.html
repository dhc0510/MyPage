**Para que sirve encadenar promesas?Encadenar promesas sirve para:Encadenar promesas en JavaScript es útil cuando tienes operaciones asincrónicas secuenciales y quieres ejecutarlas en un orden específico. Esto ayuda a evitar el anidamiento excesivo de callbacks (conocido como "callback hell") y mejora la legibilidad del código. La idea es que cada promesa se resuelva o rechace, y luego puedas encadenar la siguiente operación utilizando el resultado de la anterior.
